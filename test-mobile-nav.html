Teste de Navegação Mobile

Para testar a navegação mobile:

  1. Abra o DevTools (F12)
  2. Ative o modo de dispositivo móvel (ícone de celular)
  3. Redimensione para um tamanho mobile (ex: 375px)
  4. Clique no ícone de menu (3 linhas) no header
  5. Teste clicar nos links do menu
  6. Verifique se o menu fecha automaticamente

Funcionalidades Testadas:

  • ✅ Menu mobile abre/fecha
  • ✅ Links de navegação funcionam
  • ✅ Menu fecha automaticamente ao clicar em um link
  • ✅ Menu fecha ao clicar fora dele
  • ✅ Navegação entre páginas funciona
